# Interview React Native App

Version: 0.0.1

## Prerequisites

- Node.js >= 18
- Yarn or npm
- Android Studio
- Xcode 15 (or latest compatible with React Native 0.79.1)

## Installation

```bash
git clone git@github.com:Mbarek-Yz/emurgo-interview.git
cd interview
yarn install
yarn pods

## Installation
Android:
yarn android

iOS:
yarn ios
or
yarn iphone

```
